# Calculator Casio SL-300SV

## How to Start:

1. Clone repository
```bash
git clone https://github.com/dimagreat/mobx-calculator.git
```

2. Install dependencies

```bash
cd mobx-calculator && yarn (npm install)
```

3. Run

```bash
yarn start (npm start)
```

## Used dependencies:

* react
* mobx
* create-react-app
